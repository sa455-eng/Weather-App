# 🌦️ Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript. It fetches real-time weather data using the OpenWeather API.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- 💧 Humidity information
- 🌬️ Wind speed
- ☁️ Dynamic weather icons
- ❌ Invalid city error handling
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## 📂 Project Structure

weather-app/
│── index.html
│── style.css
│── script.js
│── images/
│── README.md

## ⚙️ How to Run

1. Clone the repository
2. Open the project in VS Code
3. Add your OpenWeather API key in script.js
4. Run index.html in your browser
